## Image Optimization

After generating all images:
1. **Compress** - Use TinyPNG.com or similar
2. **Format** - Convert to WebP for better performance (keep PNG as backup)
3. **Naming** - Use clear names: `santa-character.png`, `workshop-background.png`
4. **Organize** - Store in `/client/src/assets/images/`

**Target File Sizes:**
- Backgrounds: < 500KB each
- Characters: < 200KB each
- Icons: < 50KB each
- Logo: < 150KB
